How to retrieve your credentials

  1. Go to https://console.developers.google.com/project and create a new project.
  2. Go to APIs & Auth - APIs and enable Google+ API and Google+ Domains API
  3. Go to APIs & Auth - Credentials and click on Create new client ID. A pop-up named "Create Client ID" will appear.
  4. Select Web Application. Set Authorized JavaScript Origins to and Authorized Redirect URI to
  5. Click Create Client ID.
